This was an attempt to make a 2d isometric game in the browser.

I fairly deliberately attempted this from a naive point of view. I wanted to discover the algorithms for myself rather than implement someone else's already-discovered solution.

I really had a lot of fun building this.

I thought I would try to go simplistic first, and then optimise when needed.
